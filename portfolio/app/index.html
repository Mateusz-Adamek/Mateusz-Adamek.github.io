ABOUT ME

Obecnie z zawodu jestem Kierownikiem Projektów, jednak moje zainteresowania, wpłynęły na decyzję o nadpisaniu głównej gałęzi kariery, a w skrócie:
> git checkout –b frontend-version
> git push origin frontend-version
> git add *
Pojawiają się liczne zajęcia na początku z Java Backend, budowanie architektur systemów ITS i algorytmów na potrzeby Techniki Cyfrowej... Rodzi się fascynacja, a w następstwie:
> git commit -m "Dodano wolny czas do gałęzi frontdev-version"
> git commit -m "Dodano kurs Frontend Developer w Akademia 108 do gałęzi frontdev-version"
> git diff master..frontdev-version
> git commit -m "Gałąź frontdev-version stała się pasją i praca nad nią jest przyjemnością"
> git merge master frontdev-version
Kończąc, konfliktów nie było - posiadam ogromną motywację do rozwoju, nauki i dążę do zaimplementowania moich pasji również w karierze zawodowej.

Przechodząc do konkretów, ukończyłem intensywny kurs programowania w Akademii 108, który odbył się w miesiącach maj - lipiec 2019r.
Same przygotowania do kursu rozpocząłem już dwa miesiące wcześniej, na podstawie wykonywanych preworków czy m.in. opracowywanych przykładów udostępnianych w ramach oficjalnych dokumentacji i blogów programistycznych.
Wcześniej, jeszcze w roku 2018, amatorsko programowałem proste aplikacje backendowe wspomagając się m.in. kursami Udemy, darmowymi poradnikami dostępnymi na YouTube, a także wiedzą zdobytą jeszcze podczas warsztatów i ćwiczeń odbytych w trakcie studiów.
