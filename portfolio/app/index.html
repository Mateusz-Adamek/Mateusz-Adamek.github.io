ABOUT ME

Obecnie z zawodu jestem Kierownikiem Projektów, jednak moje zainteresowania, wpłynęły na decyzję o nadpisaniu głównej gałęzi kariery, a w skrócie:
> git checkout –b frontend-version
> git push origin frontend-version
> git add *
Pojawiają się liczne zajęcia na początku z Java Back-end, budowanie architektur systemów ITS i algorytmów na potrzeby Techniki Cyfrowej i rodzi się fascynacja, a w następstwie:
> git commit -m "Dodano wolny czas do gałęzi frontdev-version"
> git commit -m "Dodano kurs Frontend Developer w Akademia 108 do gałęzi frontdev-version"
> git diff master..frontdev-version
> git commit -m "Gałąź frontdev-version stała się pasją i jest praca nad nią jest przyjemnością"
> git merge master frontdev-version
Kończąc, konfliktów nie było - posiadam ogromną motywację do rozwoju i nauki i dążę do zaimplementowania moich pasji również w karierze zawodowej.

Jeżeli to właśnie Ty w tym roku będziesz zapraszać do siebie gości uroczystą kolację, przygotuj tę rybę tak, aby stała się jej niezapomnianym przebojem. Karp nie jest wcale trudny w oporządzeniu. Obawy przed jego sprawieniem młodsze panie domu mają tylko dlatego, że karpia przygotowuje się głównie raz w roku (od święta!), a więc rzadziej niż inne ryby. Jeżeli nigdy nie przygotowywałaś karpia lub robisz to niezwykle rzadko, przeczytaj poniższy poradnik i zastosuj się do naszych rad, a gwarantujemy, że Twoi karpolubni goście będą zachwyceni.
